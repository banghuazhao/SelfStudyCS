# Writing templates

These files mirror the upstream [**CS Self-Learning** course template](https://github.com/PKUFlyingPig/cs-self-learning/blob/master/template.md):

- `template.md` — Chinese outline  
- `template.en.md` — English outline  

They are bundled so you can open them in the app under **Course template**. To add your own guide pages to the app build, copy the filled-in Markdown into the **`Docs`** tree (see `SelfStudyCS/README.md`).
